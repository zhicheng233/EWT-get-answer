# 如何构建
1.下载 易语言 5.9 （破不破解都行自己喜欢）
2.导入 .e 源码至易语言
3.导入相关模块
4.修改、编译、发布

## 常见编译问题
Q：为毛我编译出来的大小和MD5和你发布的版本不一样
A：使用了精易助手（易语言助手）里的智能编译，编译过程中会进行脱壳压缩后再进行加壳，所有大小和MD5不同

Q：为毛我编译出来的报毒了？
A：同上一问答案
